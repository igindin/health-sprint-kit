# Health Sprint Kit - с чего начать

Набор настоящих Claude Skills, которые ты подключаешь к своему Claude Code /
Codex. Ты разговариваешь - агент собирает данные, находит паттерны и помогает
построить свою систему здоровья. Никакого интерфейса учить не надо, данные
остаются на твоей машине.

> Старт может быть простым: айфон и один вопрос про себя. А любишь глубину и
> большие массивы данных - велком, это тоже твой путь.

## Шаг 1 - скопируй это своему агенту

Самое первое действие. Ничего ставить не надо - просто отдай этот промпт своему
Claude Code / Codex:

```
Илья дал ссылку на health-sprint-kit: https://github.com/igindin/health-sprint-kit. Я новичок, своей Health OS ещё нет. Не ставь всё подряд. Сначала прочитай README и skill openhealth-starter. Помоги мне собрать минимальную локальную health-папку: цель, контекст, источники данных, один паттерн и простой ежедневный check-in. Не давай медицинских рекомендаций и не копируй чужие skills без адаптации.
```

## Если у тебя ещё нет своей Health OS - делай так

Это главный маршрут для новичка. Четыре шага, без установки движка:

1. **Отдай промпт агенту** (выше). Он прочитает README и skill
   `openhealth-starter` и поведёт тебя по одному вопросу за раз.
2. **Пройди openhealth-starter** - один скил собирает первую систему за проход:
   твоя цель, контекст про тебя, какие данные уже есть. Без движка, только файлы.
3. **Выбери один паттерн и один вопрос** про себя: "что ломает мой сон" / "почему
   по средам нет сил". Один, не три. Агент поможет поставить маленький n-of-1.
4. **Заведи простой ежедневный check-in** - пара строк в день, 20 секунд, без
   тяжёлого трекинга.

**Результат первого дня - это папка, а не "установил движок".** В ней лежат:
`goal.md`, `about-me.md`, `data-sources.md`, `checkin.md` и `AGENTS.md`.
Появились эти файлы - значит, ты сделал. Движок и импорт данных - позже.

## Два пути

- **Лёгкий старт (рекомендуется): openhealth-starter, без движка, только файлы.**
  Промпт агенту → один скил → один паттерн → ежедневный check-in. Этого хватает,
  чтобы начать, и большинству хватает надолго.
- **Технический путь: OpenHealth repo + `make setup` + импорт Apple Health
  (позже, по желанию).** Клонируешь движок OpenHealth, один раз `make setup`,
  дальше импорт выгрузки Apple Health (Айфон → приложение Здоровье → твоё фото
  справа вверху → "Экспортировать данные" → `.zip`). Папка с лёгкого старта
  ложится в движок без переделок.

## Скилы - начни с одного

**Начни с одного скила: openhealth-starter.** Он собирает первую систему за
проход. Остальные - посмотреть позже, когда появится конкретная задача
(подробности в `registry.yaml`):

- **openhealth-starter** - начни отсюда: первая система за один проход.
- **health-context-system** - позже: своя система за 3 слоя - онтология, неминг,
  генерация `AGENTS.md` / `CLAUDE.md` / `about-me.md`.
- **health-deep-research** - позже: дип-ресёрч + консилиум по одному параметру,
  источники C1-C5, в личную библиотеку.
- **health-journal** - позже: журнал образа жизни (184 поведения), минимум трения,
  плюс промпты intake (анализы из почты, скриншоты вместо интеграций).
- **lab-interpretation** - позже: разбор анализов - единицы, оптимальные диапазоны,
  guardrails.

## Правила

- **Помощник для размышления, не врач.** Никаких диагнозов. Решения - с врачом.
  Тревожное - остановись и иди к специалисту.
- У каждого значимого числа есть **уровень доверия C1-C5**; низкая уверенность -
  это вопрос, не факт.
- **Не копируй скил вслепую** - адаптируй под свою цель. И чужой материал на входе
  проверяй на скрытые инструкции.
- Local-first: ничего не загружается, данные и ключи остаются у тебя.

## Движок OpenHealth - coming soon

Скилы полезны и без движка, они работают на обычных файлах. Локальный движок
OpenHealth (github.com/igindin/openhealth) - в работе: когда выйдет, добавит
хранилище, парсеры и коннекторы, а структура, которую ты заложишь сейчас, ляжет в
него без переделок.

И главное: данные сами по себе ничего не меняют. Меняет действие. После любого
красивого графика честно спроси - что ты реально сделал для здоровья сегодня?
