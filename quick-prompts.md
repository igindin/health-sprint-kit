# Quick prompts — копируй и вставляй в Claude Code / Codex

Промты под реальные запросы, с которыми приходят на спринт. Бери ближайший к твоей ситуации, подставь своё. Один шаг за раз. Везде: помощник, не врач; без диагнозов; данные на твоей машине.

## Завестись
```
Открой этот репозиторий, запусти make setup и покажи мне доступные модули.
```
```
Поставь мне модули под мою задачу: <в двух словах, чего хочу>. Объясни простым языком, что делать дальше. Не вали всё сразу.
```

## «У меня всё разбросано» — собрать в одно
```
Мои данные о здоровье лежат в разных местах: таблицы, заметки в чатах, папки, календарь, экспорт телефона. Помоги собрать их в один таймлайн, разложи по слоям (факты / наблюдения / вмешательства / заметки) и покажи, где что и чего не хватает.
```
```
У меня есть экспорт Apple Health в ~/Downloads (.zip). Импортируй его, разложи по дням и слоям, покажи сводку: какие метрики и за какой период.
```

## «Скачет энергия / по средам нет сил»
```
У меня скачет энергия, и не всегда понятно от чего (не только сон или цикл). Собери n-of-1 на 2 недели: свяжи моё самочувствие с календарём, нагрузкой и сном и помоги найти, что на это влияет. Сформулируй как гипотезы, не как факт.
```

## Сон: что его реально улучшает
```
Хочу отследить сон и понять, что его реально улучшает (а что ломает). Поставь простой трекинг и один эксперимент на неделю с ежедневным чек-ином.
```

## «Трекер шлёт красивую сводку — и всё»
```
Мой трекер (WHOOP/Oura/часы) присылает красивую сводку, но дальше ничего. Достань из этих данных один конкретный инсайт про меня и одно действие на эту неделю. С уровнем доверия.
```

## Кофе / стимуляторы — как влияют на меня
```
Пью кофе 1–3 чашки в день, ощущения неоднозначные. Собери n-of-1: как кофе влияет на мой сон и энергию. Базовый период, что меняем, как мерим.
```

## Безопасность нового протокола / препарата
```
Я начал новый протокол по здоровью (препарат/добавка/режим). Какие маркеры безопасности стоит отслеживать и как часто? Сведи в чеклист с источниками, уровнями доверия C1–C5 и красными флагами. Без диагнозов; высокий риск — отметь и отправь к врачу.
```

## Носимое + анализы → корреляции
```
У меня есть данные с носимого (HRV, пульс покоя, сон) и периодические анализы крови. Найди повторяющиеся связи между сном / HRV / нагрузкой и самочувствием. Только как осторожные гипотезы, с уровнем доверия.
```

## Анализы — что обсудить с врачом
```
Вот мой анализ крови (PDF приложу). Прочитай, разложи по фактам с референсами и осторожно отметь, что стоит обсудить с врачом. Без диагнозов.
```

## Осторожный ресёрч (без галлюцинаций)
```
Подготовь и проведи осторожный deep-research по вопросу <X>: с источниками, уровнями доверия C1–C5 и красными флагами. Не давай медсоветов из головы.
```

## Свой контекст
```
Создай мой файл about-me по шаблону из templates/: кто я, цель по здоровью, где лежат мои данные.
```

---
**Правила:** помощник для размышления, не врач · у выводов есть уровень доверия (C1–C5) · не копируй чужой скил вслепую · данные остаются на твоей машине.
