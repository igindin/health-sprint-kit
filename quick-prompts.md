# Quick prompts — копируй и вставляй в Claude Code / Codex

Короткие готовые промты. Бери нужный, вставляй в агента. Один шаг за раз.

## Завестись
```
Открой этот репозиторий, запусти make setup и покажи мне доступные модули.
```
```
Поставь мне модули под: сон и ежедневный чек-ин. Объясни простым языком, что делать дальше.
```

## Завести данные (один источник)
```
У меня есть экспорт Apple Health в ~/Downloads (.zip). Импортируй его, разложи мои данные по слоям и покажи сводку: какие метрики и за какой период.
```
```
Вот мой анализ крови (PDF приложу). Прочитай, разложи по фактам с референсами и осторожно отметь, что стоит обсудить с врачом. Без диагнозов.
```
```
У меня мало данных. Помоги начать с того, что есть: заведём ручные заметки про сон и энергию на неделю.
```

## Один вопрос → эксперимент (n-of-1)
```
Помоги проверить один вопрос про меня: «что ломает мой сон». Сделай простой n-of-1: базовый период, что меняем, как мерим, сколько ждём. Запиши протокол. Это не диагноз.
```
```
Хочу понять, как кофе влияет на мою энергию и сон. Собери n-of-1 и поставь мне ежедневный чек-ин на 7 дней.
```

## Ежедневно
```
Сделай мой ежедневный чек-ин одним коротким вопросом и запиши ответ в таймлайн.
```
```
Покажи мои недавние инсайты с уровнем доверия и тренд по <метрике> за 30 дней.
```

## Осторожный ресёрч (без галлюцинаций)
```
Подготовь и проведи осторожный deep-research по вопросу <X>: с источниками, уровнем доверия C1–C5 и красными флагами. Не давай медсоветов из головы; высокий риск — отметь и отправь к врачу.
```

## Свой контекст
```
Создай мой файл about-me по шаблону из templates/: кто я, цель по здоровью, где лежат мои данные.
```

---
**Правила:** помощник для размышления, не врач · у выводов есть уровень доверия (C1–C5) · не копируй чужой скил вслепую · данные остаются на твоей машине.
